
# Spotify Arcade Music Tracker

Essa aplicação mostra as últimas músicas que você ouviu em um scoredboard de games arcade.



## Stack utilizada

**Front-end:** Next.js, TypeScript, StyledComponents

**Autenticação:** OAuth 2.0

**API:** Spotify
